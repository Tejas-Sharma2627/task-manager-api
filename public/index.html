User Endpoints

All user endpoints except signup and login require authentication. On signup/login, you will get a JWT.

# METHOD URL ACTION
1 POST
  1. /users
  2. /users/login
  3. /users/logoutAll
  4. /users/logout
  5. /users/me/avatar
  1. Signup (requires name, age (optional. defaults to 0), email, password)
  2. Login (requires email and password)
  3. Logout from all devices
  4. Logout
  5. Upload profile picture (under 1 MB)
2 GET
  1. /users/:id/avatar
  2. /users/me
  1. Get the profile picture (if uploaded otherwise sends an error code 404). :id takes the user id
  2. Get the user details
3 DELETE
  1. /users/me/avatar
  2. /users/me
  1. Delete user profile picture
  2. Delete user
4 PATCH
  1. /users/me
  1. Update user details

Task Endpoints

All task endpoints require authentication. You will have to use the JWT you get on signup/login.

# METHOD URL ACTION
1 POST
  1. /tasks
  1. Create a new task. (Requires description and completed (true/false))
2 GET
  1. /tasks/:id
  2. /tasks
  1. Searches a task by the task id (:id)
  2. If no query parameters given, fetches all tasks. Available query parameters:
    /tasks?completed=true
    /tasks?limit=10&skip=20
    /tasks?sortBy=createdAt:desc
3 DELETE
  1. /tasks/:id
  1. Delete a task by its id
4 PATCH
  1. /tasks/:id
  1. Update a task by its id
